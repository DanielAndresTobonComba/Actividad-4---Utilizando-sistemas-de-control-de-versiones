# TaskCards – Gestor de Tareas con Inicio de Sesión

**TaskCards** es una aplicación **frontend** que permite gestionar tareas mediante un sistema de **tarjetas interactivas**, con **inicio de sesión integrado** para mantener las tareas organizadas por usuario. Está diseñada para ser **sencilla, rápida y totalmente adaptable a diferentes dispositivos**.

---

##  Objetivo del Proyecto

El propósito de TaskCards es brindar una herramienta visual y práctica para organizar tareas diarias o proyectos personales, utilizando únicamente:

**HTML** · **CSS** · **JavaScript**  
Con persistencia de datos en el navegador mediante **LocalStorage**, sin necesidad de backend.

---

## Funcionalidades Principales

| Funcionalidad | Descripción |
|---------------|-------------|
| Inicio y cierre de sesión | Autenticación básica con localStorage |
| Creación de tarjetas de tareas | Título, descripción, fecha y estado |
| Edición de tareas | Modificación de información existente |
| Eliminación de tareas | Borrado permanente de tarjetas |
| Interfaz responsiva | Adaptación automática a PC, tablet o móvil |
